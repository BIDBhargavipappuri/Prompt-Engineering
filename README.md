# Prompt-Engineering
About Prompt Engineering

# What is prompt Engineering –

•	A prompt is natural language text that requests generative AI solutions to perform particular task.

TYPES of USERS-

Explorer – throwing quick comments to get quick answers like write email to my boss without saying th e context. We don’t get expected result; the user starts again asking questions or start editing given answer which is like using AI as search machine.

Template User – user gives requirement in 300 words in a friendly tone, with this AI can predict the use case but it’s not powerful.

The Engineer – you provide your strategy, pick an example, explain the example and provide summary, now the AI follows your logic.

With this you can achieve your expected results, save time.

Architect – architect does Chaining prompts, building agents, Automating tasks.

# Prompt engineering the art of structuring instructions so AI gives Accurate, creative, context driven results instead of hallucination or random guesses.

# Now Types of Prompt Engineering – 

1)	Zero Shot prompt – no examples are given; AI works on given sentence. When you don’t have examples, it is for straight forward task. AI uses general knowledge to response.

2)	One shot Prompt – 1 example is given and it guides the AI to provide O/p. It’s like showing someone exactly how to do task once to avoid confusion and saving time. This type is good for accurate O/P without brainstorming multiple examples.

3)	Few shot prompt – multiple examples increase efficiency of O/p.
4)	COT – Chain of Thoughts prompting – breaking & explaining the instructions. To show the work in steps.
5)	React Prompt(Reasoning + Acting) - thinking step by step and acting like calling a tool, searching, or performing a sub task.
  6) Role prompting- assign role like as you are developer, teacher.
  7) Contextual prompting- Adding background info, examples for accurate and relevant responses. 
  8) multi model/hybrid prompting- combining any 2 or 3 of above.
  9) Negative prompts- explicitly telling AI what to avoid. 

By using right prompts, LLM’s evolve as autonomous planners. With this we can enter in to world of AI agents.
